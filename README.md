# Robotic-Arm

### 6 eksenli, 4 modüler başlığı olan, lise ve üniversite öğrencilerine yönelik eğitsel robot kol.

* ATmega2560
* Stepper Motor (Nema17, Nema23)
* Motor Driver(TMC2225, TB6560, A4988)
* TTL / USB Debugging Board
* ESP 32
